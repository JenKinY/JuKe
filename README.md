# JuKe-For Android
# 九星创客商品分销系统
## 一、描述
使用AndroidStudio开发的一款商品分销系统，业务流程根据九星创客设计。  
- 开发环境:  
    JDK 1.8  
    AndroidStudio V3.4.1  
- 后端数据服务：
    Bmob云数据库(bmob-sdk:3.7.0)+Bmob云函数
